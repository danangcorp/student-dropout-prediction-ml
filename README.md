# 🎓 Student Dropout Prediction using Machine Learning

This project predicts the risk of student dropout using machine learning models.

## Models Used

* Logistic Regression
* Random Forest
* Gradient Boosting

## Best Model

Logistic Regression
AUC-ROC: **0.821**

## Features

* GPA
* CGPA
* Attendance Rate
* Stress Index
* Study Hours

## Deployment

This model is deployed using **Streamlit**.

## Run Locally

```bash
streamlit run app.py
```
